# Model Selection and Evaluation

## Project Overview

This project compares multiple machine learning models using the Titanic Dataset to determine the best-performing model for passenger survival prediction.

## Objective

To evaluate and compare different machine learning algorithms using various performance metrics and cross-validation techniques.

## Dataset

Titanic Dataset

Target Variable:

* Survived

Features Used:

* Pclass
* Sex
* Age
* SibSp
* Parch
* Fare
* Embarked

## Models Used

1. Logistic Regression
2. Decision Tree Classifier
3. Random Forest Classifier

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* Cross Validation Accuracy

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

## Results

The models were trained and evaluated on the Titanic dataset. Their performance was compared using multiple evaluation metrics.

Random Forest achieved the best overall performance and was selected as the final model.
